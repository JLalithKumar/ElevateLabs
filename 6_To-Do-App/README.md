# Java Swing To-Do App

A simple **desktop GUI application** to manage tasks using Java Swing. The application allows users to **add, delete, and view tasks** dynamically in a user-friendly interface.

---

## Objective
To create a Java program that allows users to **maintain a To-Do list** using a graphical user interface (GUI). The project introduces **Java Swing components** such as `JFrame`, `JButton`, `JTextField`, and `JList` while providing hands-on experience with desktop GUI development.

---

## Features
- **Add Task** – Input a task in the text field and add it to the list.  
- **Delete Task** – Remove a selected task from the list.  
- **View Tasks** – Display all tasks in a scrollable list.  
- **User-Friendly GUI** – Simple interface with buttons and input fields.  

---

## Implementation Details
- **ToDoApp Class** – Contains the main method and GUI implementation.  
- **JFrame** – Main window of the application.  
- **JTextField** – Input field to enter new tasks.  
- **JButton** – Buttons to add or delete tasks.  
- **JList & DefaultListModel** – Displays tasks and maintains them dynamically.  
- **ActionListener** – Handles button click events for adding and deleting tasks.  
- **Layout Management** – `BorderLayout` and `JPanel` used to arrange components neatly.  

---

## Conclusion
This project provides a practical example of **Java Swing GUI programming**. It demonstrates how to design a **simple interactive desktop application**, manage dynamic data with `DefaultListModel`, and handle user actions via event listeners. Users learn the basics of **desktop GUI development** in Java in a structured and engaging way.
